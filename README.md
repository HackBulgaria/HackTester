#### In case you want to run the documentation locally:


-   Build the documentation:

<!-- -->
    $ mkdocs build

-   Run your documentation server:

<!-- -->
    $ mkdocs serve

-   Go to `localhost:8000` or run the whole hacktester project and go to `/docs/` and see the documentation.
