#### In case you want to read the documentation locally:

```
$ cd /hacktester
```

-   Build the documentation:

<!-- -->
    $ mkdocs build

-   Run your documentation server:

<!-- -->
    $ mkdocs serve

-   Go to `localhost:8000` and see the documentation.
